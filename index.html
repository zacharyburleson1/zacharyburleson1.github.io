About

I graduated from Southern New Hampshire University with a Bachelor's degree in Information Technology. While I was getting my degree, I fell in love with the wonderfull world of web development. From backend coding to frontend design, I enjoy the entire process, and it's always exciting to see how everything comes together to create an enjoyable user experience.

I’m always looking to improve, and I’m eager to learn more and more about web development! My current skill set is HTML5, CSS3, JavaScript, and Bootstrap. I also have some experience on personal coding projects using things like C++, Java, and SQL. My favorite editor to use at the moment is Visual Studio Code.
