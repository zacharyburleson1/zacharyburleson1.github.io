Simple Weather App

    This Weather Search project is a modified version of a tutorial I found that was made by George Martsoukos (George’s tutorial can be found here). It was an interesting project where I learned a lot about APIs and AJAX. Since there are so many APIs out there that can add so much to a webpage, I was excited to learn more about them in this weather project. Plus I learned more about AJAX, and how we can fetch data and update a portion of the webpage, instead of having to reload the entire page every time there’s a change. I’m glad I worked on this project, since it opens the door to so many possibilities!
